# Projeto_CyberSecurity_Dio

## 📄 Introdução
Este projeto tem como objetivo demonstrar, de forma prática, a captura de eventos de teclado e a aplicação de criptografia em arquivos utilizando Python. Para isso, são utilizadas as bibliotecas pynput e cryptography, permitindo explorar conceitos importantes relacionados à interação com o sistema operacional e à segurança da informação.

A aplicação integra dois principais componentes: um responsável pelo monitoramento e registro das teclas pressionadas pelo usuário em tempo real, e outro voltado à criptografia de arquivos dentro de um diretório específico. Dessa forma, o projeto aborda temas como manipulação de entrada, tratamento de exceções, escrita em arquivos e uso de algoritmos criptográficos.

Com foco educacional, este projeto permite compreender na prática como eventos de entrada podem ser capturados e como dados podem ser protegidos por meio de criptografia, contribuindo para o desenvolvimento de habilidades em programação e segurança da informação.

## Codigo Ramsoware
Este código implementa um mecanismo de criptografia de arquivos utilizando a biblioteca cryptography, especificamente o algoritmo Fernet, que fornece criptografia simétrica segura.

O programa percorre um diretório, criptografa os arquivos encontrados e gera uma mensagem informativa ao usuário. Sequestrando os arquivos e podendo ser descriptografado apenas com a chave criada pelo código.

## Código Keylogger
Este código utiliza a biblioteca pynput para capturar eventos de teclado em tempo real de um computador infectado e registrar as teclas pressionadas em um arquivo de texto chamado log.txt. O programa cria um listener de teclado que fica em execução contínua, monitorando todas as teclas pressionadas pelo usuário. Cada tecla capturada é tratada e registrada no arquivo de log de forma apropriada.
